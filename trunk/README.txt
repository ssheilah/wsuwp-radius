Use a shortcode with three attributes to embed an iframe containing a Washington State University Hobsons Radius contact form. 

== Description ==
A plugin using a shortcode with three attributes to embed an iframe containing a Hobsons Radius contact form. 

The shortcode accepts three attributes: 
1. URL (the URL of the form; must contain proper host -- wsuonline.hobsonsradius.com -- and end in .scc). The URL is verified using a regular expression.
2. Form title. A default title will appear if none is specified.
3. Height (an integer). This should vary with the form length and may need to be adjusted after the form is inserted on the page.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the \'Plugins\' screen in WordPress
